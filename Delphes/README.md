## Getting Started
1. Clona el repositorio:
   ```bash
   git clone https://github.com/castaned/simulation-lab-course
   cd simulation-lab-course
   ```
2. Descarga los datos simulados Z->ee 
   ```bash
   wget http://cp3.irmp.ucl.ac.be/downloads/z_ee.hep.gz
   gunzip z_ee.hep.gz
   ```
